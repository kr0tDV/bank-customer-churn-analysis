# Bank Customers Analysis

Анализ данных клиентов банка для выявления факторов оттока, сравнения клиентов по странам и выделения наиболее ценных клиентов.  
Добавлено консольное приложение для прогнозирования оттока нового клиента на основе обученной модели.

---

## 📊 О датасете

**Источник:** [Bank Customer Churn Dataset (Kaggle)](https://www.kaggle.com/datasets/saurabhbadole/bank-customer-churn-dataset)  
**Содержание:** данные о 10 000 клиентах банка. Каждая запись содержит демографические, финансовые и поведенческие характеристики клиента, а также метку – вышел клиент (Exited = 1) или остался (Exited = 0).

**Основные поля:**
- `CustomerId` – уникальный идентификатор клиента
- `Surname` – фамилия
- `CreditScore` – кредитный рейтинг
- `Geography` – страна проживания (Франция, Германия, Испания)
- `Gender` – пол
- `Age` – возраст
- `Tenure` – количество лет, в течение которых клиент состоит в банке
- `Balance` – баланс счета
- `NumOfProducts` – количество банковских продуктов, используемых клиентом
- `HasCrCard` – наличие кредитной карты (0/1)
- `IsActiveMember` – активность клиента (0/1)
- `EstimatedSalary` – предполагаемая зарплата
- `Exited` – целевая переменная (1 – отток, 0 – остался)

---

## 🧹 Обработка данных

Проведена предварительная очистка:
- Удалены строки с пропусками (их не оказалось).
- Удалены дубликаты по `CustomerId` (уникальный идентификатор).
- Очищенные данные сохранены в `data/processed/`.

---

## 🔍 Анализ

### 1. Сравнение клиентов по странам
Построены распределения числовых и категориальных признаков для Франции, Германии и Испании. Сохранены таблицы и графики в `result/geography_analysis/`.

### 2. Анализ оттока (churn)
- Сравнены характеристики клиентов, которые ушли и остались.
- Построены boxplots для числовых признаков и stacked bar charts для категориальных.
- Обучены модели логистической регрессии и случайного леса для прогнозирования оттока.
- Оценена точность и важность признаков.
- Результаты сохранены в `result/churn_analysis/`.

### 3. Выделение топ-10 самых ценных клиентов
- Отобраны действующие клиенты (`Exited = 0`).
- Отсортированы по балансу (убывание), затем по активности и зарплате.
- Сформирован список для персонализированных предложений.
- Результат сохранен в `result/top_active_customers/`.

### 4. Консольное приложение для прогнозирования оттока
Разработан скрипт `predict_churn.py`, который:
- Загружает обученную модель и предобработчики.
- В интерактивном режиме запрашивает данные клиента.
- Выдаёт прогноз: останется клиент или уйдёт, а также вероятность оттока.
- Даёт рекомендации по действиям.

## ▶️ Как запустить

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/ВАШ_ЛОГИН/bank-customer-churn-analysis.git
   cd bank-customer-churn-analysis
   ```

2. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. **Очистка данных и обучение модели** (необходимо один раз):
   ```bash
   python scr/main.py                     # очистка данных
   python scr/churn_analysis.py           # анализ оттока и обучение модели
   ```
   После этого в папке `models/` появятся файлы модели и предобработчиков.

4. **Запуск консольного приложения для прогнозов**:
   ```bash
   python scr/predict_churn.py
   ```
   Следуйте инструкциям в консоли – введите данные клиента и получите прогноз.

5. **Другие скрипты** (опционально):
   ```bash
   python scr/geography_analysis.py       # анализ по странам
   python scr/top_active_customers.py     # топ-10 клиентов
   ```

6. **Jupyter Notebook**:
   Запустите `report.ipynb` для просмотра полного анализа и визуализаций.

## 📌 Основные выводы

- Клиенты из разных стран различаются по кредитному рейтингу, балансу, активности.
- На отток сильнее всего влияют возраст, активность, баланс и количество продуктов.

- Выделены 10 клиентов с высоким балансом, активностью и зарплатой для персональных предложений.
